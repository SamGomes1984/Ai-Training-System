# AI Training System

This is an AI training system that allows users to select pre-trained models, upload data, and train the models in a memory-efficient way using AutoMem and AutoScaler.

## Requirements

- Python 3.8+
- Streamlit
- PyTorch
- Hugging Face Transformers
